# Arches AI

Small edge AI engines, tuned on top of the arches.

Arches AI is a workshop for local intelligence: compact systems that watch, listen, decide, and help without needing the cloud for every breath.

## Projects

- [Arches Watch](https://github.com/arches-labs/arches-watch) - A fast multi-camera watchtower for RTSP streams, built to stay sharp as camera count grows.
